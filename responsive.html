2.4.1. What is Responsive Design?

Responsive design is a web design technique that allows web pages to efficiently adapt to different screen sizes and devices, such as desktop computers, tablets, and mobile phones. The goal is to provide a consistent and optimal user experience, regardless of the device used to access the website.

2.4.2. Media Queries in CSS

Media queries are a key CSS3 feature that allows applying different CSS styles depending on device or browser window characteristics, such as size, orientation, or resolution.

Basic syntax:
@media (condition) {
  /* CSS styles that will apply if the condition is met */
}

2.4.3. Common Conditions in Media Queries

1. max-width and min-width: These conditions allow applying CSS styles depending on the maximum or minimum browser window size.

2. orientation: Allows applying styles based on device orientation, either landscape (horizontal) or portrait (vertical).

3. resolution: Used to adjust styles based on device resolution, such as high pixel density screens (e.g., Retina).

2.4.4. Using Media Queries for Responsive Design

The main idea is to adjust your design for different devices using min-width and <-width> within media queries. In this example:

  • If the screen is wider than 768px, default styles apply (Light Gray background).
  • If the screen is 768px or narrower (like tablets), styles inside the media query apply (Light Blue background).
  • If the screen is 480px or narrower (like mobile), corresponding styles apply (Light Yellow background).

(The image border turns green and width becomes 100% on high-resolution screens)
