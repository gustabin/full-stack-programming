5.1. What is AJAX?

Asynchronous JavaScript and XML

AJAX is a technique that allows web applications to send and receive data from a server asynchronously without reloading the page.

No Page Reload

Update specific parts without refreshing

Faster

Improved speed and user experience

Efficient

5.1. AJAX with jQuery

jQuery AJAX Methods

jQuery simplifies AJAX requests with $.ajax(), $.get(), $.post()

  • GET Get Posts
  • POST Create Post
Example: jQuery AJAX
$.ajax({
  url: "https://jsonplaceholder.typicode.com/posts",
  type: "POST",
  data: JSON.stringify({ title: "Test", body: "Content" }),
  contentType: "application/json",
  success: function(response) { console.log(response); },
  error: function(xhr, status, error) { console.error(error); }
});

5.4. Handling Responses and Errors

HTTP Status Codes & Error Handling

✅ Success Codes
  • 200 OK - Successful request
  • 201 Created - Resource created
  • 204 No Content - Success, no data
❌ Error Codes
  • 400 Bad Request - Client error
  • 401 Unauthorized - Not authorized
  • 404 Not Found - Resource not found
  • 500 Internal Server Error - Server error
Test Error Handling
Error Handling with Fetch API
fetch(url)
  .then(response => {
    if (!response.ok) {
      throw new Error(`Error ${response.status}`);
    }
    return response.json();
  })
  .catch(error => console.error(error));

5.2. RESTful API in PHP (Reference)

Backend API Structure

Example of a PHP RESTful API that manages users in MySQL:

users.php - API Endpoint

<?php
header("Content-Type: application/json");
include "db.php";

$method = $_SERVER["REQUEST_METHOD"];
switch ($method) {
  case "GET": getUsers($mysqli); break;
  case "POST": addUser($mysqli); break;
  case "PUT": updateUser($mysqli); break;
  case "DELETE": deleteUser($mysqli); break;
}

function getUsers($mysqli) {
  $result = $mysqli->query("SELECT * FROM users");
  $users = $result->fetch_all(MYSQLI_ASSOC);
  echo json_encode($users);
}
?>
Testing with cURL
GET Users:
curl -X GET http://localhost/api/users.php

POST User:
curl -X POST http://localhost/api/users.php \
  -H "Content-Type: application/json" \
  -d '{"name": "John", "email": "john@mail.com"}'

PUT User:
curl -X PUT http://localhost/api/users.php \
  -H "Content-Type: application/json" \
  -d '{"id": 1, "name": "John Doe"}'

DELETE User:
curl -X DELETE http://localhost/api/users.php \
  -H "Content-Type: application/json" \
  -d '{"id": 1}'
