2.6.1. What is FontAwesome?

FontAwesome is one of the most popular and widely used icon libraries in web development. It provides a vast variety of vector icons that can be used on websites and applications.

Scalable

Icons can be resized without losing quality

Customizable

Change colors, sizes, and animations with CSS

Extensive

Thousands of icons in various categories

2.6.2. & 2.6.3. Including FontAwesome

Two Common Methods

Via CDN (Fastest)

Add this to the <head> of your HTML:

<link href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.3/css/all.min.css" rel="stylesheet">
Local Download

Download from FontAwesome.com and link locally:

<link href="path/to/fontawesome/css/all.min.css" rel="stylesheet">

2.6.10. Icons with Links

Social Media Buttons

Icons can be used as interactive links, perfect for social media buttons:

Facebook Twitter Instagram LinkedIn GitHub
Code Example

  <i class="fab fa-facebook fa-3x"></i>
</a>
